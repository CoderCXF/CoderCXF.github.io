《私房菜》学会了几道菜?

发表于 2020-05-18 分类于 Linux OS 阅读次数： Valine：

最近在读《鸟哥的Linux私房菜》发现真的不错，不愧是 Linux 方面的经典书籍，非常适合用来入门和扫盲。推荐就完事了…

书中都讲了什么？

首先这本书是真的很厚，很多人看到这本书之后就会由衷的来一句：卧槽，经典，以后再来拜读， 其实大家真的都是这样，因为实在是太厚了，900多页将近1000页。但是最近闲在家里就一时兴起觉得先看看难度怎样，结果就一发不可收拾，立马虚拟机、Centos 安排上了。

这本书确实很厚，但是讲的东西其实是有点扫盲性质的，全篇都在讲 Linux 的基础和一些常用的命令，比如 Linux 中的目录和文件的权限和管理，打包压缩和解压等等操作，大家对其中的很多东西都是有概念的。除此之外，还有就是一些在 Linux 操作系统中必会的一些工具，比如有 vim 编辑器 、bash 、shell script 和 rpm 包管理工具等等，所以总的来做这本书真的很好学习，尤其是可以一边看书一边实践 ( 写指令 )，学习进度真的非常快。

来梳理一下书中所介绍大概内容：

  1. 介绍 Linux 和 Linux distributions,以及如何学习 Linux,这一章基本就是扫盲；
  2. Linux 的磁盘分区和为在虚拟机上安装 Linux 做铺垫；
  3. 在 KVM 上安装 Centos 7.1 的流程并且进行相关的配置说明；当然我们可以在 VMWare 等虚拟机软件上安装都可以；后面有一章节介绍的是专门对 Centos 的基础设置，例如：日期和时间、语系和防火墙设置等等；
  4. Linux 首次登陆以及对 X window 模式和文字模式的介绍，还有一些基础指令，最重要的就是 --help 和 man page 的使用，大家多试几次就可以很容易理解和掌握；
  5. Linux 文件和目录权限，整个一章都是在讲那十个字符 -/d/l/b/c rwx-rwx-rwx，以及三个改变文件属性和权限的命令： chgrp、chown 和 chmod；
  6. 介绍对文件和目录的常用操作命令： cd、pwd、mkdir 和 rmdir 以及 cp、rm和 mv；以及查看文件的命令： cat、tac、head、tail、more、less等常见操作，都是基础命令；
  7. 磁盘与文件系统的管理
  8. 文件与文件的打包压缩和解压常用的命令，比如 tar 指令：
  9. vim 程序编辑器，就是一个编辑器，和 VSCode 一样，学习也就是学习一些常用的快捷键操作，比如：快速定位到某一行 nG，快速定位到行首 0 和行尾 $，删除行dd 或者是 ndd ( 删除第n行 )，复制粘贴等等常用操作快捷键；
  10. 介绍 bash 和基本的使用方法以及基本的使用命令；
  11. 正则表达式，正则表达式就是处理字符串的方法；主要介绍了基础正则和延伸的正则；
  12. Shell script ，shell script 是利用 shell 的功能所写的一个“程序 （program）”，这个程序是使用纯文本文件，将一些 shell 的语法与指令（含外部指令）写在里面， 搭配正则表达式、管线命令与数据流重导向等功能，以达到我们所想要的处理目的；介绍的也仅仅是入门级别的shell script;
  13. 如何在 Linux 中安装软件以及卸载软件两大方法；以及 rpm (yum) 和 srpm 两个包管理工具；
  14. 还介绍了 X window 以及设置；

Tips: 总的来说好像东西看着挺多，确实是这样，但是仔细的划分一下，其实并不算多，除了我们几个工具： vim bash shell script、rpm 和 srpm，其余的都是操作系统的基础知识和 Linux 的常用命令，所以这本书可以当做手册来用。哪里不会翻哪里！

下面只总结了 vim bash 和shell script 的基础知识。

因为书中内容太多，所以还是建议买来当做参考书使用！

第二道菜：bash

1. 什么是 bash 和 shell

首先要搞明白的就是什么是 shell?
shell 从字面意思理解就是 壳，在操作系统中指的是与操作系统的核心进行交互的接口，使用者可以通过应用程序指挥操作系统的核心，然后再让核心去操作硬件以完成我们的需求。
也就是说，只要能够操作应用程序的接口都能够称为壳程序。狭义的壳程序指的是指令列方面的软件，比如大名鼎鼎的 bash 就是一种 shell。shell 就是 Linux 的任督二脉。

那么什么又是 bash 呢? bash 其实就是 shell 的一种, shell 不是特定的一种程序,我们可以理解为是一个广义的概念，有好几家 shell 程序。
而 Linux 默认使用的是 bash, 所以能学好 bash 就很了不起了。

第三道菜：shell script

shell script 是利用 shell 的功能所写的一个“程序 （program）”，这个程序使用纯文本文件，将一些 shell 的语法与指令（含外部指令）写在里面， 搭配正则表达式、管线命令与数据流重导向等功能，以达到我们所想要的处理目的。

简单的来说就是将一些指令写在一个文件中放在一起执行。

而且 shell script 更提供阵列、 循环、条件与逻辑判断等重要功能，让使用者也可以直接以 shell 来撰写程序,可以像写高级语言一样撰写程序。

2.多重判断

if [ 条件判断式 ]; then

当条件判断式成立时，可以进行的指令工作内容；

fi


if [ 条件判断式一 ]; then

当条件判断式一成立时，可以进行的指令工作内容；

elif [ 条件判断式二 ]; then

当条件判断式二成立时，可以进行的指令工作内容；

else 当条件判断式一与二均不成立时，可以进行的指令工作内容；

fi
